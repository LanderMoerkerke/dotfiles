## Requirements

-   Window manager: [Hyprland](https://github.com/hyprwm/Hyprland) (Wayland)
-   Status bar: waybar
-   Notifcation manager: dunst
-   Terminal: foot
-   Shell: zsh
-   Editor: nvim
-   Image viewer: imv
-   Browser: qutebrowser
-   File manager: lf

## Other

-   Alternative Xorg window manager: [dwm](https://github.com/LanderMoerkerke/dwm)
